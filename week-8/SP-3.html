React Project Setup using Create React App

This page demonstrates the steps to set up a React project using CRA.

Steps:

  1. Install Node.js and npm from the official site: Node.js
  2. Open terminal or command prompt
  3. Run the command to create a new React app: 
    npx create-react-app my-app
  4. Navigate into the project folder: 
    cd my-app
  5. Start the development server: 
    npm start

Verify:

Open http://localhost:3000 in a browser. You should see the React default welcome page.
